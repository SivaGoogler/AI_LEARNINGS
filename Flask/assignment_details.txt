Flask assignment

This is a simple RESTful Math API built with Flask.

Details:
- GET / - Health check endpoint that returns available endpoints.
- POST /add - Adds two numbers (num1 + num2).
- POST /subtract - Subtracts two numbers (num1 - num2).
- POST /multiply - Multiplies two numbers (num1 * num2).
- POST /divide - Divides two numbers (num1 / num2). Returns error if num2 is 0.

All endpoints except "/" require a JSON body with "num1" and "num2" fields.

Run:
python3 Flask/assignment.py

The app will start on http://localhost:5000

Requirements:
- flask
- Python 3

Testing:
Use Postman or curl to test the endpoints. See README.md for detailed Postman instructions.
